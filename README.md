## Description
This project is a simplified student enrollment system. We use a **RESTful API with Spring Boot and Java**. 
The main object is to show the mapping between entities with JPA.


![Main model](https://erp.emanuelcosta.com.br/img-sistemas/java/enrollment/1-main-model.png)


The first thing to do is configure your database credentials in the **application.properties** file.
